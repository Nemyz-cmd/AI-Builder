# Project Blueprint

## Overview

This project is a lottery number generator.

## Current Implementation

The application is a simple "Hello, world!" page.

## Plan

- **index.html:**
    - Set the title to "Lotto Number Generator".
    - Add a heading for the title.
    - Add a button to generate lottery numbers.
    - Add a container to display the generated numbers.
- **style.css:**
    - Style the lottery number generator, including the button and the number display.
- **main.js:**
    - Implement the logic to generate 6 unique random numbers between 1 and 45.
    - Display the generated numbers on the screen.
